## Packages
framer-motion | For complex page transitions, staggered reveals, and smooth animations
@react-three/fiber | For the 3D particle background and floating heart effects
@react-three/drei | Helpers for Three.js (OrbitControls, Stars, Text)
three | Core 3D library required by fiber/drei
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to efficiently merge Tailwind CSS classes

## Notes
The application features a heavy 3D background using Three.js.
Animations are critical for the "cinematic" feel.
Colors are strictly defined in index.css variables for consistency.
